## Testing

Make sure you have [Node](http://nodejs.org) installed and then install [Mocha](http://visionmedia.github.io/mocha/) either globally:

```
	node install -g mocha
```

... or locally:

```
	node install
```

Then just run:

```
	node test
```
or:

```
	node test/test.js
```
